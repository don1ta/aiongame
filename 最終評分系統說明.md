# AION 遊戲角色評分系統說明

## 📊 評分系統總覽（總分 100 分）

本系統旨在量化玩家角色的綜合實力，透過五大維度進行評分。

### 權重分配

| 項目 | 權重 | 計算方式 | 說明 |
| :--- | :---: | :--- | :--- |
| **🔥 裝備品階** | **30分** | 指數加權 | 反映裝備強度 (含古文石強化) |
| **📋 板塊數量** | **15分** | 難度加權 | 依據板塊取得難度給分 |
| **🐾 寵物理解度** | **20分** | 線性計算 | 寵物培養廣度 |
| **⚔️ 技能烙印** | **30分** | 分段式計算 | 核心目標與極限追求 |
| **🏅 稱號數量** | **5分** | 線性計算 | 稱號收集數量 |
| **總計** | **100分** | | |

## 🔥 1. 裝備品階 (30分)

採用 **指數加權系統**，讓高強化的頂尖裝備價值更突出。

### 品階基礎分

| 品階 | 顏色 | 基礎分 |
| :--- | :--- | :---: |
| **神話** | <span style="color:#e67e22">█ 橘色</span> | **10.0** |
| **傳說** | <span style="color:#f1c40f">█ 金色</span> | **7.0** |
| **史詩** | <span style="color:#3498db">█ 藍色</span> | **4.5** |
| **特殊** | <span style="color:#00ffcc">█ 青色</span> | **2.5** |
| **稀有** | <span style="color:#2ecc71">█ 綠色</span> | **1.5** |
| **普通** | <span style="color:#bdc3c7">█ 白色</span> | **0.5** |

### 計算公式

單件裝備得分由四部分組成：

```math
單件得分 = 基礎分 + 強化加成 + 突破加成 + 閃耀加成
```

1.  **強化加成**：`基礎分 × (強化等級 / 20)^1.2`
2.  **突破加成**：`基礎分 × (突破等級 / 5)^1.5`
3.  **閃耀加成**：若為閃耀裝備，額外 `+20% 基礎分`

### 分數換算
*   **單件滿分參考**：約 **32分** (神話+20, 突破+5, 閃耀)
*   **最終分數**：將全身所有裝備的得分加總後，換算為 **30分** 制。
    *   *註：目標總強度約 500 分可拿滿 30 分。*

## 🐾 3. 寵物理解度 (20分)

採用 **線性計算**，鼓勵多樣化培養。

*   **公式**：`(理解度種類 / 8) × 20`
*   **說明**：共 8 種理解度，每解鎖一種約得 2.5 分。
*   **滿分**：擁有 8 種理解度。

## ⚔️ 4. 技能烙印 (30分)

採用 **分段式目標積分 (Piecewise Scoring)**，鼓勵達成核心目標並保留極限追求。

### (1) 強度換算 (Intensity Point)
針對前 12 強技能，依等級給予強度分：

*   **Lv.20** (特化IV): **100 強度**
*   **Lv.15** (特化III): **75 強度**
*   **Lv.10** (特化II): **45 強度**
*   **Lv.5** (特化I): **20 強度**
*   **Lv.1-4**: **5 強度**

### (2) 分段計算公式
根據總強度 (Max 1200) 計算最終得分 (Max 30)：

*   **核心階段 (0-400 強度)**：
    *   目標：達成 4 個核心技能 Lv.20 (400 強度)。
    *   權重：此階段佔總分 **80% (24分)**。
    *   **公式**：`(總強度 / 400) × 24`
*   **極限階段 (400-1200 強度)**：
    *   目標：追求極限滿分。
    *   權重：此階段佔總分 **20% (6分)**。
    *   **公式**：`24 + ((總強度 - 400) / 800) × 6`

### (3) 分數對照參考
*   **285 強度 (混搭)**：**17.1 分** (核心目標達成率 71%)。
*   **400 強度 (4個Lv.20)**：**24.0 分** (S級，核心目標達成)。
*   **800 強度 (8個Lv.20)**：**27.0 分** (SS級)。
*   **1200 強度 (全滿)**：**30.0 分** (SSS級，完美畢業)。

## 🏅 5. 稱號數量 (5分)

採用 **線性計算**。

*   **公式**：`(擁有數量 / 400) × 5`
*   **滿分**：擁有 400 個稱號。
